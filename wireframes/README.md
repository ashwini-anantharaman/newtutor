# Owlwise Wireframes

Visual wireframes for the Owlwise platform — **use the Figma files below** (not the HTML folder; those were an early draft).

## Figma files (copy into your design)

| Workflow | Screens | Open in Figma |
|----------|---------|---------------|
| **Student Workflow** | 15 frames, stacked top-to-bottom | [Student Workflow Wireframes](https://www.figma.com/design/1D242nAx5mSotx7BnB3Dmu) |
| **Instructor Workflow** | 13 frames, stacked top-to-bottom | [Instructor Workflow Wireframes](https://www.figma.com/design/0xMIGUgslX39hjiC4ftE9u) |

Both files are in your Figma **drafts** (Ashwini Anantharaman's team).

### How to copy into your design file

1. Open the workflow file in Figma.
2. Scroll down the canvas — each screen is labeled **SCREEN 01 · LANDING**, etc.
3. Select a frame (e.g. `01 — Landing`) and **⌘C / Ctrl+C**.
4. Open your design file and **⌘V / Ctrl+V**.
5. Repeat for each screen in order.

Each frame is **1440×956** (artboard + screen label bar) with labeled sidebar, content blocks, and annotations matching the built app.
